# wanted-pre-onboarding-fe 선발 과제
원티드의 프리온보딩 프론트엔드 코스 선발 과제입니다. Todo List를 작성하고, 제공해주신 API서버와의 데이터 CRUD를 구현했습니다.


## 구현 영상
